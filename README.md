# Quantum Kaleidoscope

An interactive visual meditation and relaxation tool featuring kaleidoscopic patterns, breathing exercises, and binaural audio.

## Features

- **Kaleidoscope Visualization**: Dynamic, symmetrical patterns with customizable segments and color themes
- **Journey Presets**: Quick (5min), Deep (20min), Gamma (10min), and Psychedelic (15min) modes
- **Breathing Pacer**: Multiple breathing modes including Box breathing, 4-7-8, and Heart Coherence
- **Gamma Entrainment**: 40Hz visual and audio stimulation options
- **Audio Reactive**: Responds to microphone input or uploaded music files
- **Binaural Beats**: True binaural audio for headphone users

## Usage

Simply open `index.html` in a modern web browser, or visit the deployed version.

### Keyboard Shortcuts
- **Space**: Pause/Play
- **F**: Toggle Fullscreen
- **B**: Toggle Breathing Pacer
- **←/→**: Adjust Speed

## Deployment

This is a static HTML application that can be deployed to any static hosting service.

### Deploy to Render
1. Push to GitHub
2. Create a new Static Site on Render
3. Connect your GitHub repository
4. Set publish directory to `/` (root)

## Warning

⚠️ This application contains flashing lights and pulsing patterns that may trigger seizures in people with photosensitive epilepsy.
